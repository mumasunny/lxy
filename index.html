小红书作品解析

本页只请求 XHS-Downloader 自带的接口 POST /xhs/detail（默认 127.0.0.1:5556），与 Vue FastAPI Admin 无关。
请先在 XHS-Downloader 目录执行：python main.py api。
浏览器跨域限制：若直接双击打开本 HTML 并请求 5556 端口会被拦截。请在本目录运行 python serve.py，再用浏览器打开提示的地址（通过本地代理转发，避免跨域）。

视频下载失败 / 只有低清？请先完成下面两项
  1. 必须填 Cookie（视频尤其重要）：未设置 Cookie 时，官方接口往往只给低清或不稳定。请用下方「如何复制 Cookie」步骤，把浏览器里小红书网页版的整段 Cookie 粘贴进来（需包含 web_session 等字段）。
  2. 代理：只有在你本机 Clash / V2 等已开启且端口一致时才填（如 http://127.0.0.1:7890）。若代理没开或填错，会导致取不到地址或下载失败——请先清空代理再试；项目说明里也提到全局代理可能导致异常。

详细图文见： XHS-Downloader → Cookie 说明

如何复制 Cookie（简要）
  1. Chrome/Edge 打开 https://www.xiaohongshu.com/explore（可先登录也可不登录）。
  2. 按 F12 →「网络」→ 勾选「保留日志」→ 筛选里输入 cookie-name:web_session。
  3. 在页面里随便点开一条笔记，在网络面板点任意一条请求，在「请求标头」里找到 Cookie，整段复制。
  4. 粘贴到本页下方「小红书 Cookie」文本框，再勾选下载并重试。
